全栈工程师项目

FULL STACK ENGINEER

项目介绍

本次项目将开发一款类似于抖音/小红书的短视频全栈 App(iOS & Android)， 帮助学生熟悉互联网产品的全程开发， 包括前端以及 后端开发， 以及项目上线， 对求职软件工程师和全栈工程师都有极强的背景补充和技能提升作用。

  • React Native
  • Typescript
  • Node.js
  • Express.js
  • MongoDB
  • Mongoose
  • JWT
  • AWS S3
  • AWS Elastic Beanstalk
  • AWS Cognito
  • Apifox

适合学员

对 Javascript 有基本了解， 适合 CS 科班出身以及希望转行 CS 并就业 SDE 的同学

项目跨度以及时长

4 个月 | 40 小时： 后端 20 hours + 前端 20 hours

授课方式

理论知识 + 项目设计开发 + 独立编码 + office hour + 导师评估
